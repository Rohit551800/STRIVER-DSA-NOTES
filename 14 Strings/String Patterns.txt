-----------------------------------------------------
PATTERN 1: HASHING / FREQUENCY PATTERN
-----------------------------------------------------

Ye pattern kya hota hai?
Jab question me ye words dikhe:

- Count batao
- Compare karo
- Kitni baar aaya
- Anagram check karo

Tab frequency array ya map use hota hai.

-----------------------------------------------------
Example Question:
Check if two strings are Anagram

s = "listen"
t = "silent"
-----------------------------------------------------

Solution Idea:
Dono strings ke har character ki frequency same honi chahiye.

Code Logic (C++):

int freq[26] = {0};

// s ke characters count karo
for(char c : s)
    freq[c - 'a']++;

// t ke characters subtract karo
for(char c : t)
    freq[c - 'a']--;

// check karo
for(int i = 0; i < 26; i++)
    if(freq[i] != 0)
        return false;

return true;

-----------------------------------------------------
Steps (jab bhi Hashing pattern aaye):
-----------------------------------------------------
1. Decide character range (a-z / A-Z / ASCII)
2. Frequency array ya map banao
3. String traverse karo
4. Count / compare karke answer nikalo

-----------------------------------------------------
PATTERN 2: TWO POINTER PATTERN
-----------------------------------------------------

Ye pattern kya hota hai?
Jab:

- Start vs End compare karna ho
- Reverse karna ho
- Palindrome check karna ho

Tab do pointers use hote hain (left, right).

-----------------------------------------------------
Example Question:
Check Palindrome

s = "madam"
-----------------------------------------------------

Code Logic:

int l = 0, r = s.size() - 1;

while(l < r) {
    if(s[l] != s[r])
        return false;
    l++;
    r--;
}

return true;

-----------------------------------------------------
Steps (Two Pointer):
-----------------------------------------------------
1. Ek pointer start pe rakho
2. Ek pointer end pe rakho
3. Compare / swap karo
4. Pointers move karte raho

-----------------------------------------------------
PATTERN 4: SLIDING WINDOW / SUBSTRING
-----------------------------------------------------

Ye pattern kya hota hai?
Jab question bole:

- Longest / shortest substring
- Without repeating
- Window maintain karo

-----------------------------------------------------
Example Question:
Longest Substring Without Repeating Characters
-----------------------------------------------------

Code Logic:

unordered_set<char> st;
int l = 0, ans = 0;

for(int r = 0; r < s.size(); r++) {

    while(st.count(s[r])) {
        st.erase(s[l]);
        l++;
    }

    st.insert(s[r]);
    ans = max(ans, r - l + 1);
}

return ans;

-----------------------------------------------------
Steps:
-----------------------------------------------------
1. Window start aur end rakho
2. Duplicate aaye to window shrink karo
3. Condition valid ho to expand karo
4. Answer update karte raho
