1 HASHING / FREQUENCY PATTERN
📌 Ye pattern kya hota hai?
Count batao
Compare karo
Kitni baar aaya
Anagram check karo
👉 Tab frequency array / map use hota hai
✅ Example Question
Check if two strings are Anagram
s = "listen"
t = "silent"
🔍 Solution Idea
Dono strings ke har character ki frequency same honi chahiye
int freq[26] = {0};

// s ke characters count karo
for(char c : s)
    freq[c - 'a']++;

// t ke characters subtract karo
for(char c : t)
    freq[c - 'a']--;

// check karo
for(int i = 0; i < 26; i++)
    if(freq[i] != 0)
        return false;

return true;
🧠 Steps (Hashing Pattern)
1️⃣ Decide character range (a-z / A-Z / ASCII)
2️⃣ Frequency array banao
3️⃣ String traverse karo
4️⃣ Count / compare karke result nikalo

4 SLIDING WINDOW / SUBSTRING
📌 Ye pattern kya hota hai?
Longest / shortest substring
Without repeating
Window maintain karo
✅ Example Question
Longest Substring Without Repeating Characters
unordered_set<char> st;
int l = 0, ans = 0;

for(int r = 0; r < s.size(); r++) {

    // Duplicate mila? Shrink window
    while(st.count(s[r])) {
        st.erase(s[l]);
        l++;
    }

    st.insert(s[r]);
    ans = max(ans, r - l + 1);
}

return ans;
🧠 Steps
1️⃣ Window start aur end rakho
2️⃣ Duplicate aaye to window shrink karo
3️⃣ Condition valid ho to expand karo
4️⃣ Answer update karte raho
